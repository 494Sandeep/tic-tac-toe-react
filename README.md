# Tic Tac Toe (React)

A simple Tic Tac Toe game built with React and TypeScript.

## Features

- Two-player mode (Player 1 vs Player 2)
- Visual turn indicator
- Win and draw detection
- Board reset functionality
- Responsive UI with Tailwind CSS

## Getting Started
